WARBL2 Configuration Tool

Compiled by Stuart Hertzog

2024-06-02

1 Introduction

WARBL Icon

The Configuration Tool is a web page that connects to the WARBL2 using MIDI protocol to let you customize its Settings. Because the Configuration Tool uses the relatively new Web MIDI protocol, It is necessary to use a browser that supports Web MIDI, which currently include:

  • Chrome (v43+)
  • Firefox (needs Jazz Web MIDI Extension)
  • Microsoft Edge (v79+), and
  • Opera
  • Safari unfortunately does not yet support Web MIDI.

on

  • Windows
  • Mac
  • Linux
  • Chromebook, or
  • Android

On iOS devices you can use the free WARBL app, generously made by Michael Eskin of Appcordions.

1.1 How To Use This Site

The Table of Contents on the left is scrollable:

  • Mouse into this area and use your Mouse Scroll Button or the List Position Slider to move up and down the list
  • Click on any heading to go directly to that Page or Section

The small Hamburger icon at the top left of the Displayed Page is clickable:

  • Click it to Show/Hide the Table of Contents column on the left.
